# ⚙ First Time Setup

## Step 1

Install Python 3.13

Verify

```
py -3.13 --version
```

---

## Step 2

Install Google Chrome

Verify

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --version
```

---

## Step 3

Run Installer

```
install.bat
```

Ye automatically check karega

- Python
- pip
- Chrome
- Selenium
- ChromeDriver
- Folder Structure

---

## Step 4

Run Test

```
run.bat
```

Agar

```
Home | Mynaukri
```

open ho jaye

to setup successful hai.
